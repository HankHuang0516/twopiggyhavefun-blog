<p><span style="color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">門司燒咖哩一直收藏在我的口袋名單，餐廳交通方式非常方便~</span></p>
<p><span style="font-size: 12pt; color: #000000;">靠近捷運國父紀念館站附近、延吉街巷弄中，身為起司控的人當然不能錯過!!</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160503_224237_574(1)" src="https://pimg.1px.tw/lolwarden/1463222579-278732284.jpg" alt="IMG_20160503_224237_574(1)" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">店內空間不是很大，平日晚上用餐人潮也蠻多，還好有事先訂位嘿嘿&nbsp;</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160503_224355_087" src="https://pimg.1px.tw/lolwarden/1463222579-3121860113.jpg" alt="IMG_20160503_224355_087" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">廚房區前排的座位區，吧檯設計充滿居酒屋的味道</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160503_224404_259" src="https://pimg.1px.tw/lolwarden/1463222579-1939027058.jpg" alt="IMG_20160503_224404_259" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">值得介紹的是，這裡的點餐方式是使用點餐機~</span></p>
<p><span style="font-size: 12pt; color: #000000;">先在點餐機上點好餐後，會有一張點餐紙印出來，再將紙交給店員才算點餐完畢唷</span></p>
<p><span style="font-size: 12pt; color: #000000;">咖哩分成五種口味：浪人咖哩、聖女咖哩、戀愛咖哩、男子漢咖哩、歇斯底里咖哩</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160503_224338_118" src="https://pimg.1px.tw/lolwarden/1463222579-2392598217.jpg" alt="IMG_20160503_224338_118" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">第一次用餐使用點餐機點餐，算是一個很特別的體驗</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160503_224348_208" src="https://pimg.1px.tw/lolwarden/1463222579-1154275253.jpg" alt="IMG_20160503_224348_208" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">點餐機旁還有放置咖哩飯小碗、大碗的範本，供大家參考</span></p>
<p><span style="font-size: 12pt; color: #000000;">但個人是覺得小碗、大碗份量好像沒有差很多，點個小碗就覺得吃蠻飽的XD</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160503_224342_304" src="https://pimg.1px.tw/lolwarden/1463222579-3041223059.jpg" alt="IMG_20160503_224342_304" border="0"> &nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>特製薯條 $60</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">看似普通的薯條，口感非常酥脆，會讓人一根接一根停不下來餒</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224330_812" src="https://pimg.1px.tw/lolwarden/1463223875-1571033463.jpg" alt="IMG_20160503_224330_812" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>門司三鮮捲 $50</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">吃起來有點類似雞捲，雖然份量不是很多，但內餡蠻扎實的，充滿濃郁的海鮮香氣~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224304_082" src="https://pimg.1px.tw/lolwarden/1463223875-1324385796.jpg" alt="IMG_20160503_224304_082" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>百憂解雞翅 $50</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">為何稱作百憂解雞翅呢?!其實這也讓我百思不解XD</span></p>
<p><span style="font-size: 12pt; color: #000000;">烤至金色焦黃的雞翅，可以吃出是先醃漬後才烤的，口感吃起來偏甜，瞬間有種豁然開朗的感覺<img src="//s.pixfs.net/f.pixnet.net/images/emotions/teeth_smile.gif" alt="" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224334_011" src="https://pimg.1px.tw/lolwarden/1463223875-3714345208.jpg" alt="IMG_20160503_224334_011" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;"><span style="color: #ff6600;"><strong>聖女咖哩(小盛) 生蛋加起司</strong> $120+40 (附湯)</span></span></p>
<p><span style="font-size: 12pt; color: #000000;">因為我本身不太敢吃辣，就點了唯一不辣的聖女咖哩!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">聖女咖哩是用北海道奶油塊燉煮，真的一點都不辣唷</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224123_567" src="https://pimg.1px.tw/lolwarden/1463223875-968117554.jpg" alt="IMG_20160503_224123_567" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">身為蛋黃控+起司控，當然兩種都要加上囉<img src="//s.pixfs.net/f.pixnet.net/images/emotions/teeth_smile.gif" alt="" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224138_799" src="https://pimg.1px.tw/lolwarden/1463223875-2519758919.jpg" alt="IMG_20160503_224138_799" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">聖女咖哩攪拌後，咖哩味道不是很明顯，反而奶香味比較濃厚~</span></p>
<p><span style="font-size: 12pt; color: #000000;">裡面還會加入幾塊雞肉，口感相當軟嫩!!</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224231_916" src="https://pimg.1px.tw/lolwarden/1463223875-1419247697.jpg" alt="IMG_20160503_224231_916" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>浪人咖哩(大盛)</strong>&nbsp;<strong>生蛋加起司</strong>&nbsp;$160+40<strong>&nbsp;(附湯)</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">男友點的浪人咖哩，帶著淡淡的蘋果香氣，上面加上滿滿的起司， 看起來超讚的餒<img src="//s.pixfs.net/f.pixnet.net/images/emotions/embaressed_smile.gif" alt="" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224113_180" src="https://pimg.1px.tw/lolwarden/1463223875-164482522.jpg" alt="IMG_20160503_224113_180" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">雖然說是生蛋，但是<span style="font-family: Helvetica, Arial, 'LiHei Pro', PMingLiU, sans-serif; line-height: 26px;">蛋白的部分都有熟，吃起來滑</span><span style="font-family: Helvetica, Arial, 'LiHei Pro', PMingLiU, sans-serif; line-height: 26px;">滑嫩</span><span style="font-family: Helvetica, Arial, 'LiHei Pro', PMingLiU, sans-serif; line-height: 26px;">嫩的口感</span></span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224119_903" src="https://pimg.1px.tw/lolwarden/1463223875-514714602.jpg" alt="IMG_20160503_224119_903" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">試吃了一口男友點的浪人咖哩，由於辣度過重，舌頭味蕾都被辣麻痺了~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">讓平常不吃辣的我真的無法再吃下第二口~哈哈</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160503_224126_581" src="https://pimg.1px.tw/lolwarden/1463223875-2707573591.jpg" alt="IMG_20160503_224126_581" border="0"></span></p>

<p><span style="font-size: 12pt; color: #000000;">整體而言，&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">門司燒咖哩不僅交通方便，</span></p>
<p><span style="font-size: 12pt; color: #000000;">更讓人覺得耳目一新的是，</span></p>
<p><span style="font-size: 12pt; color: #000000;">點餐方式是使用點餐機，</span></p>
<p><span style="font-size: 12pt; color: #000000;">讓顧客有個特別的體驗~</span></p>
<p>&nbsp;</p>
<p><span style="font-size: 12pt; color: #000000;">這家不只專賣咖哩，</span></p>
<p><span style="font-size: 12pt; color: #000000;">還可以嘗到好幾種不同風味的咖哩，</span></p>
<p><span style="font-size: 12pt; color: #000000;">像是浪人咖哩、戀愛咖哩、歇斯底里咖哩、男子漢咖哩及聖女咖哩，</span></p>
<p><span style="font-size: 12pt; color: #000000;">如果像我一樣不吃辣的人，可以點聖女咖哩，</span></p>
<p><span style="font-size: 12pt; color: #000000;">濃郁的口感帶著淡淡的咖哩香氣，</span></p>
<p><span style="font-size: 12pt; color: #000000;">味道也相當溫和，沒有像浪人咖哩充滿辣勁~</span></p>

<p><span style="font-size: 12pt; color: #000000;">門司燒咖哩真的是一間很有特色的專賣店，</span></p>
<p><span style="font-size: 12pt; color: #000000;">不論是充滿奶香的聖女咖哩、口味較重的浪人咖哩，都有各自特色~</span></p>
<p><span style="font-size: 12pt; color: #000000;">對於熱愛吃咖哩、起司控的人來說，這家會是不錯的選擇唷~</span></p>
