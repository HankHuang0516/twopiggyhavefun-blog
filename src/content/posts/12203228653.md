<p><span style="font-size: 12pt;"><span style="letter-spacing: 1px; line-height: 20px; color: #0000ff; font-family: 新細明體;">* 喜歡旅遊、</span><span style="letter-spacing: 1px; line-height: 20px; color: #0000ff; font-family: 新細明體;">愛吃美食嗎？歡迎加入&nbsp;<span style="color: #777777; text-decoration: underline;"><a style="color: #000000; text-decoration: underline;" href="https://www.facebook.com/twopiggyhavefun/?fref=ts" target="_blank"><span style="color: #ff0000; text-decoration: underline;">兩隻小豬</span><span style="color: #ff0000; text-decoration: underline;">粉絲團</span></a></span>**</span></span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #0000ff; font-family: 新細明體; font-size: 12pt;"><img title="page" src="https://pimg.1px.tw/lolwarden/1464604426-739656932.jpg" alt="page" width="570" height="456" border="0">&nbsp;&nbsp;</span></p>
<p><span style="font-size: 12pt;"><strong><span style="color: #993300;">泰街頭</span></strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">地址：台北市大安區溫州街74巷6號1樓</span></p>
<p><span style="font-size: 12pt; color: #000000;">電話：02-2367-6997</span></p>
<p><span style="font-size: 12pt; color: #000000;">營業時間：11:30–20:00</span></p>
<p><span style="font-size: 12pt; color: #000000;">臉書<span style="text-decoration: underline; color: #0000ff;"><a href="http://www.facebook.com/thaijietou/" target="_blank"><span style="color: #0000ff; text-decoration: underline;">請點這</span></a></span></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">悶熱的天氣，總會讓人想吃些酸辣的食物~</span></p>
<p><span style="font-size: 12pt; color: #000000;">既然沒有機會去到泰國，那就來這家嘗嘗道地的泰國街頭美食!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">這次跟大家介紹很厲害的【<strong><span style="color: #993300;">泰街頭</span></strong>】<img src="//s.pixfs.net/f.pixnet.net/images/emotions/032.gif" alt="" border="0">，<span style="font-family: 新細明體;">簡直是<span style="line-height: 30.6px;">把路邊攤開在餐廳裡~</span></span></span></p>
<p><span style="font-size: 12pt; color: #000000;">走在溫州街上，很難發現這是一家泰式料理，不說還以為是咖啡店呢XD</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7006" src="https://pimg.1px.tw/lolwarden/1464604522-3037528504.jpg" alt="IMG_7006" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">泰街頭的店名其實很容易記，其實就是「<strong><span style="color: #993300;">泰國街頭美食</span></strong>」的縮寫</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7007" src="https://pimg.1px.tw/lolwarden/1464604522-887908856.jpg" alt="IMG_7007" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7008" src="https://pimg.1px.tw/lolwarden/1464604522-283340745.jpg" alt="IMG_7008" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">店內採半開放式廚房、吧檯座位，當天平日晚餐時間來用餐，人潮絡繹不絕阿~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">而且店家員工不多，看著前方這位員工忙進忙出，真是辛苦!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">特別的是，店內是使用藍色、紅色的塑膠桌椅，真的很像在吃路邊小吃的感覺耶XD</span></p>
<p><span style="font-size: 12pt; color: #000000;">眼尖的朋友們應該會發現架上有放好幾罐阿華田，當下我內心也充滿問號??</span></p>
<p><span style="font-size: 12pt; color: #000000;">回家查了資料，原來店內的"街頭雞肉丸子豬雜湯"裡面的滷汁有加了「<strong><span style="color: #993300;">阿華田</span></strong>」，超酷的啦!!</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img style="font-size: 12px;" title="IMG_7013" src="https://pimg.1px.tw/lolwarden/1464604522-3263095825.jpg" alt="IMG_7013" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #993300;"><strong>泰街頭MENU</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">主要以泰國街頭風味為主，有烤雞腿飯、咖哩包、月亮蝦餅、摩摩喳喳、泰式奶茶等</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20160519_215541_459" src="https://pimg.1px.tw/lolwarden/1464604522-4045631061.jpg" alt="IMG_20160519_215541_459" border="0"> &nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>泰東北烤雞腿飯 $150</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">來到泰街頭，最讓我期待就是他的烤雞腿飯，看起來超道地的阿~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">泰東北烤雞腿飯配菜有大黃瓜、涼拌木瓜絲、辣酸菜、溏心蛋，並附上泰式辣醬自行沾取</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7014" src="https://pimg.1px.tw/lolwarden/1464604629-1845521425.jpg" alt="IMG_7014" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">表皮烤至焦黃的雞腿飯，香氣十足~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7016" src="https://pimg.1px.tw/lolwarden/1464604629-4184109216.jpg" alt="IMG_7016" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;"><span style="color: #ff0000;"><strong>雞腿份量真的超大!!!!!</strong></span>完全把白飯蓋住，還要先把雞腿默默移開吃白飯XD</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7015" src="https://pimg.1px.tw/lolwarden/1464604629-1309047892.jpg" alt="IMG_7015" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">烤雞腿飯送上桌時，已經先將雞腿切成四大塊))超誘人~~&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">雞腿有著炭烤香氣，肉質超鮮嫩多汁，搭配白飯超級好吃!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">烤雞腿本身不會辣，即使不沾醬也好好吃唷~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7018" src="https://pimg.1px.tw/lolwarden/1464604629-971850765.jpg" alt="IMG_7018" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>鮮蝦起司咖哩飯 $160</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">這道一送上桌時，很難不注意他的盤子耶XD超有泰國風味~</span></p>
<p><span style="font-size: 12pt; color: #000000;">亮眼紅色的大白盤，上面盛裝鮮蝦起司醬和白飯，看起來超美味~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7025" src="https://pimg.1px.tw/lolwarden/1464604629-2879988501.jpg" alt="IMG_7025" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">"鮮蝦起司咖哩"除了有鮮蝦，還有紅蘿蔔、馬鈴薯~</span></p>
<p><span style="font-size: 12pt; color: #000000;">咖哩醬汁配上白飯口感滑順，混著起司讓每一口充滿奶香四溢，</span></p>
<p><span style="font-size: 12pt; color: #000000;">再搭配紅蘿蔔與馬鈴薯很清爽解膩，但後勁會有點微辣XD</span></p>
<p><span style="font-size: 12pt; color: #000000;">雖然主角明蝦只有一隻，卻佔了咖哩一半的空間，但肉質相當Q彈呢~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7027" src="https://pimg.1px.tw/lolwarden/1464604629-3569363303.jpg" alt="IMG_7027" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>月亮蝦餅 $120</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">每次吃泰式必點的月亮蝦餅，總共有8片左右~</span></p>
<p><span style="font-size: 12pt; color: #000000;">外皮炸得相當酥脆，還可以吃到裡面的蝦肉丁，再沾上泰式酸辣醬超讚啦~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">因為真的太好吃，回家馬上跟姐姐分享這家，姐姐說月亮蝦餅$120元也太便宜!!!!我也要吃XD</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7029" src="https://pimg.1px.tw/lolwarden/1464608735-423586987.jpg" alt="IMG_7029" border="0"> &nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>奶咖啡(冰) $60</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">造型超可愛的奶咖啡，小鐵桶裡面裝著咖啡牛奶、全脂鮮乳，還有結凍後的咖啡塊</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7019" src="https://pimg.1px.tw/lolwarden/1464604629-1262446775.jpg" alt="IMG_7019" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">泰式奶咖啡的喝法非常有趣，將鮮奶和咖啡牛奶混合倒入結凍後的咖啡塊，就變成奶咖啡囉~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7021" src="https://pimg.1px.tw/lolwarden/1464604629-4025289555.jpg" alt="IMG_7021" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">有別於一般直接調好的奶咖啡，自己動手調比例更有趣~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7022" src="https://pimg.1px.tw/lolwarden/1464604629-4149159810.jpg" alt="IMG_7022" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>美斯樂紅頂奶茶(冰) $50</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">雖然幾乎一整杯都是冰塊， 但奶茶味道還是很濃郁喔</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_7024" src="https://pimg.1px.tw/lolwarden/1464604629-54297928.jpg" alt="IMG_7024" border="0">&nbsp;&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">隱藏在溫州街裡的巷弄，竟然有著一家泰式餐廳，</span></p>
<p><span style="font-size: 12pt; color: #000000;">不只賣泰國街頭美食，更把泰國路邊攤搬進餐廳裡，讓店內氛圍真的很"街頭"</span></p>
<p>&nbsp;</p>
<p><span style="font-size: 12pt; color: #000000;">餐點最讓我滿意的就是"泰東北烤雞腿飯"，</span></p>
<p><span style="font-size: 12pt; color: #000000;">吃過一次後，到現在仍讓我意猶未盡，加上我本身是肉食主義啦哈哈XD</span></p>
<p><span style="font-size: 12pt; color: #000000;">再來，更讚的就是月亮蝦餅，不但價格平價，還相當好吃~</span></p>
<p>&nbsp;</p>
<p><span style="font-size: 12pt; color: #000000;">當天用餐客人大多是學生、剛下班的上班族們，</span></p>
<p><span style="font-size: 12pt; color: #000000;">外面還有好幾組客人在候位，代表這家人氣不錯唷~</span></p>
<p><span style="font-size: 12pt; color: #000000;">愛吃泰式料理的朋友們絕對不能錯過嘿嘿~~</span></p>
