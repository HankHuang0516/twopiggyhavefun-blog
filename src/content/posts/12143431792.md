<p><span style="color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">來到貓空，除了要搭乘超夯的貓纜，還要來泡茶聊天看風景!!</span></p>
<p><span style="font-size: 12pt;"><span style="color: #000000;">這次兩隻小豬來到這家<strong><span style="color: #ff0000;">正大休閒茶園</span></strong>~</span></span></p>
<p><span style="color: #000000;"><span style="font-size: 12pt;">坐到貓空纜車後再步行3分鐘就到嚕&nbsp;</span></span></p>
<p><span style="font-size: 12pt; color: #000000;">正大休閒茶園菜單種類非常豐富，有經濟合菜、茶類、茶點...<br></span></p>
<p><span style="font-size: 12pt; color: #000000;">我們兩人點了一杯冰紅茶、冷泡高山茶、起司蛋糕</span></p>
<p><span style="font-size: 12pt; color: #000000;">因為當天天氣實在太熱了~就來喝點冰涼的</span></p>
<p><span style="font-size: 12pt; color: #000000;"><span style="font-size: 16px;">冷泡高山茶他是裝在一個玻璃容器，中間空心的地方裝冰塊，更加保冰(超酷的!!)</span></span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151013_154702_831" src="https://pimg.1px.tw/lolwarden/1444797432-2993484904.jpg" alt="IMG_20151013_154702_831" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">冰紅茶喝起來普通，起司蛋糕一口咬下非常綿密，香草冰淇淋超順口~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151013_154706_812" src="https://pimg.1px.tw/lolwarden/1444797432-2610409580.jpg" alt="IMG_20151013_154706_812" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">餐點價格都不貴~冰塊還可以續加，一直喝一直喝XD</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151013_154710_438" src="https://pimg.1px.tw/lolwarden/1444797432-681945805.jpg" alt="IMG_20151013_154710_438" border="0"></span></p>
<p><span style="font-size: 16px; color: #000000;">正大休閒茶園有兩層樓，我們坐到2樓的露天雅座</span></p>
<p><span style="font-size: 12pt; color: #000000;">雖然當天天氣非常熱，不過坐在2樓吹著自然風，蠻涼爽的</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151013_154747_851" src="https://pimg.1px.tw/lolwarden/1444797432-1514848294.jpg" alt="IMG_20151013_154747_851" width="407" height="545" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">其實坐在2樓有個好處就是可以看到美景!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">包括台北101、信義區或是台北盆地都盡收眼底~<img src="//s.pixfs.net/f.pixnet.net/images/emotions/heart.gif" alt="" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151013_154754_983" src="https://pimg.1px.tw/lolwarden/1444797432-755861828.jpg" alt="IMG_20151013_154754_983" width="417" height="558" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">休息完後，就來排隊搭貓纜嚕</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151013_155429_084" src="https://pimg.1px.tw/lolwarden/1444797432-1603491805.jpg" alt="IMG_20151013_155429_084" width="383" height="511" border="0"> &nbsp;</span></p>
<p><span style="color: #000000;">&nbsp;<img style="font-size: 16px;" title="IMG_20151013_155409_460" src="https://pimg.1px.tw/lolwarden/1444797432-604290696.jpg" alt="IMG_20151013_155409_460" width="379" height="506" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">正大休閒茶園整體還說，餐點價格都很平價，非常適合學生族群聚餐</span></p>
<p><span style="font-size: 12pt; color: #000000;">而且風景也不錯，下次有機會的話想來晚上，美景一定很美~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">想來貓空聊天泡茶卻不曉得要到哪一家嗎?</span></p>
