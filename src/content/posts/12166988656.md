<p><span style="font-size: 12pt; color: #000000;">即將邁入秋冬季節，當然少不了一鍋暖呼呼的羊肉爐~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">今天跟大家介紹這家<span style="color: #ff0000;"><strong>羴(ㄕㄢ)大王羊肉爐吃到飽</strong></span>!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">位於新店北宜路上的新店國小旁，靠近捷運新店站</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_223156" src="https://pimg.1px.tw/lolwarden/1446715382-670110606.jpg" alt="IMG_20151101_223156" border="0"> &nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">大人只要399元，孩童99公分以下免費，100公分~129公分200元，130公分以上399元</span></p>
<p><span style="font-size: 12pt; color: #000000;">一人開鍋499元，飲酒清潔費100元，浪費食材需加收費用喔</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203923_625" src="https://pimg.1px.tw/lolwarden/1446702169-189170168.jpg" alt="IMG_20151101_203923_625" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">這家共有兩層樓，我們假日下午5點多到，被安排坐到2樓座位</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203747_901" src="https://pimg.1px.tw/lolwarden/1446702169-3448427926.jpg" alt="IMG_20151101_203747_901" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">2樓座位看起來很大，外面還能看到風景，不過黑漆漆的((笑</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203904_212" src="https://pimg.1px.tw/lolwarden/1446702169-3069020703.jpg" alt="IMG_20151101_203904_212" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">店家真是佛心來著，提供免費WI-FI((歡呼</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203837_815" src="https://pimg.1px.tw/lolwarden/1446702169-1374391729.jpg" alt="IMG_20151101_203837_815" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">兩隻小豬認為~吃到飽的好處就是可以夾自己喜歡吃的料((嘻嘻</span></p>
<p><span style="font-size: 12pt; color: #000000;">不過要酌量取用，才不會浪費食物唷</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203757_087" src="https://pimg.1px.tw/lolwarden/1446702169-1193414073.jpg" alt="IMG_20151101_203757_087" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">這裡有各種香菇都吃的到噢，包含大香菇、蘑菇、金針菇....</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203805_083" src="https://pimg.1px.tw/lolwarden/1446702169-4253095395.jpg" alt="IMG_20151101_203805_083" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">還有鴨血、羊肚，架上看起來剩不多了~~應該大受好評</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203814_753" src="https://pimg.1px.tw/lolwarden/1446702169-3082653343.jpg" alt="IMG_20151101_203814_753" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">還有對身體健康的番茄、木耳</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203822_181" src="https://pimg.1px.tw/lolwarden/1446702169-1930087258.jpg" alt="IMG_20151101_203822_181" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">香菇下面就是帶皮羊肉啦~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203830_191" src="https://pimg.1px.tw/lolwarden/1446702169-3877271417.jpg" alt="IMG_20151101_203830_191" border="0"> &nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">還有王子麵，連張君雅小妹妹也有哈哈</span></p>
<p><span style="font-size: 12pt; color: #000000;">一看到冬粉，立馬夾了好幾個，個人覺得寬冬粉超好吃，非常Q彈唷~~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203712_115" src="https://pimg.1px.tw/lolwarden/1446704540-943932808.jpg" alt="IMG_20151101_203712_115" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">麻辣牛腩及白飯，不過我們沒有裝，想留點胃吃羊肉~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203740_039" src="https://pimg.1px.tw/lolwarden/1446704540-61508875.jpg" alt="IMG_20151101_203740_039" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">這邊的羊肉湯、蔬菜湯是自己裝唷，請酌量取用~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203632_436" src="https://pimg.1px.tw/lolwarden/1446704271-1119607902.jpg" alt="IMG_20151101_203632_436" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">蔥花、羊肉沾醬也是自取，羊肉沾醬還分成辣跟不辣，</span></p>
<p><span style="font-size: 12pt; color: #000000;">因為我肉喜歡吃原味的，就沒有沾醬歐</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203647_465" src="https://pimg.1px.tw/lolwarden/1446704271-1635750991.jpg" alt="IMG_20151101_203647_465" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">在拍照的時候，老闆娘走過來說這邊飲料都是自己煮的!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">養生牛蒡茶喝起來味道不會奇怪，口感非常清爽~</span></p>
<p><span style="font-size: 12pt; color: #000000;">阿薩姆紅茶味道濃厚，是我喜歡的味道~~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203724_097" src="https://pimg.1px.tw/lolwarden/1446704540-4232773801.jpg" alt="IMG_20151101_203724_097" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">店家非常強調，全面禁煙!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">浪費食材，剩餘過多將以一兩30元計價酌收費用，自行帶走~</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203621_584" src="https://pimg.1px.tw/lolwarden/1446704270-3546791640.jpg" alt="IMG_20151101_203621_584" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">雖然2樓沒有飲料機，但是有一壺紅茶可以享用唷</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203657_881" src="https://pimg.1px.tw/lolwarden/1446704271-81188332.jpg" alt="IMG_20151101_203657_881" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">老闆娘真的非常善待我們，怕我們下去1樓裝太麻煩，直接端上一盤羊肉來((好開心</span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;<img title="IMG_20151101_203506_403" src="https://pimg.1px.tw/lolwarden/1446704116-3459977534.jpg" alt="IMG_20151101_203506_403" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">我們夾了超多金針菇，真的很新鮮好好吃</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203513_955" src="https://pimg.1px.tw/lolwarden/1446704116-155338135.jpg" alt="IMG_20151101_203513_955" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">吃羊肉爐，除了要控制好份量，還要注意食材比例搭配，才能吃得健康又盡興唷！</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203535_445" src="https://pimg.1px.tw/lolwarden/1446704116-3387351458.jpg" alt="IMG_20151101_203535_445" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">小豬夾超多豆皮，根本是豆皮控XD而我是冬粉控?!</span></p>
<p><span style="font-size: 12pt; color: #000000;">不得不說，他們冬粉真的超好吃，尤其是寬冬粉，加上羊肉湯一起吃，口感一級棒</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203550_160" src="https://pimg.1px.tw/lolwarden/1446704116-425433374.jpg" alt="IMG_20151101_203550_160" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">耶~下料後準備開吃囉~~肉跟菜都吃到了，好滿足啊!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">羊肉湯非常暖胃又暖心，我裝了好幾碗喝，最後羊肉還吃不下XD</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203606_041" src="https://pimg.1px.tw/lolwarden/1446704116-1177598020.jpg" alt="IMG_20151101_203606_041" border="0"> &nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">羊肉非常大塊、肉相當軟嫩，一點羊騷味也沒有呢~</span></p>
<p><span style="font-size: 12pt; color: #000000;">我們裝了好幾盤吃哇哈哈</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203915_655" src="https://pimg.1px.tw/lolwarden/1446704540-2375474479.jpg" alt="IMG_20151101_203915_655" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">因為太好吃了，小豬表情才這麼沉重嗎?!</span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;<img title="IMG_20151101_203857_485" src="https://pimg.1px.tw/lolwarden/1446704540-2027379622.jpg" alt="IMG_20151101_203857_485" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">羊肉不只吃到飽，連冰淇淋也是唷</span></p>
<p><span style="font-size: 12pt; color: #000000;">最後胃真的撐到不行，就沒吃到冰淇淋囉</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203704_117" src="https://pimg.1px.tw/lolwarden/1446704271-1131900914.jpg" alt="IMG_20151101_203704_117" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">口味有很多種，有瑞士巧克力、花生、紅豆牛奶、芝麻</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203640_311" src="https://pimg.1px.tw/lolwarden/1446704271-4060706609.jpg" alt="IMG_20151101_203640_311" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">還有巧克力、香草、香芋、草莓</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img title="IMG_20151101_203732_216" src="https://pimg.1px.tw/lolwarden/1446704540-3687321555.jpg" alt="IMG_20151101_203732_216" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">老闆娘的熱情款待，讓我們吃得好開心~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">羴大王羊肉爐很多人都會將「<strong>羴</strong>」念錯，正音教學一下~這是念<strong><span style="color: #ff0000;">ㄕㄢ</span></strong>!!</span></p>
<p><span style="font-size: 12pt; color: #000000;">這家食材都非常新鮮，不用怕腸胃會不舒服，而且羊肉湯頭非常鮮甜~ &nbsp;</span></p>
