<p><span style="font-size: 12pt;"><span style="letter-spacing: 1px; line-height: 20px; color: #0000ff; font-family: 新細明體;">**喜歡旅遊、</span><span style="letter-spacing: 1px; line-height: 20px; color: #0000ff; font-family: 新細明體;">愛吃美食嗎？歡迎加入<span style="color: #ff0000;">&nbsp;<span style="text-decoration: underline;"><a style="color: #000000; text-decoration: underline;" href="https://www.facebook.com/twopiggyhavefun/?fref=ts" target="_blank"><span style="color: #ff0000; text-decoration: underline;">兩隻小豬粉絲團</span></a></span></span>**</span></span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_223444_801" src="https://pimg.1px.tw/lolwarden/1467719220-169951496.jpg" alt="IMG_20160703_223444_801" border="0"></span></p>
<p><strong><span style="font-size: 12pt; color: #0000ff;">坪林貴妃池</span></strong></p>
<p><span style="font-size: 12pt; color: #000000;">交通路線:&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">於坪雙公路(北42鄉道)往闊瀨營地方向約13公里處的小粗坑橋上游溪谷，距離闊瀨營地不到</span><span style="font-size: 12pt; color: #000000;">1公里，左邊會看到土地公廟，即可停車</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img style="color: #0000ff; font-family: 新細明體; letter-spacing: 1px; line-height: 20px;" title="IMG_20160703_224043_880" src="https://pimg.1px.tw/lolwarden/1467719220-887306192.jpg" alt="IMG_20160703_224043_880" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">坪林的路途有點彎曲，連在機車上都還有點暈車<img src="//s.pixfs.net/f.pixnet.net/images/emotions/whatchutalkingabout_smile.gif" alt="" border="0"></span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_224130_285" src="https://pimg.1px.tw/lolwarden/1467719220-215519886.jpg" alt="IMG_20160703_224130_285" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">經過跋山又涉水，路途破百公里，終於來到傳說中的私房景點「貴妃池」啦！&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">這裡的池水分為上下兩池，長徑各約五、六公尺，</span></p>
<p><span style="font-size: 12pt; color: #000000;">像是一座迷人的天然大澡盆，因而被取名為「貴妃池」</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_224221_975" src="https://pimg.1px.tw/lolwarden/1467719220-1383688381.jpg" alt="IMG_20160703_224221_975" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_223513_388" src="https://pimg.1px.tw/lolwarden/1467719220-1295112010.jpg" alt="IMG_20160703_223513_388" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">7月份來這裡真的很適合~</span></p>
<p><span style="font-size: 12pt; color: #000000;">不只有冰涼清澈的泉水，還有高聳的樹蔭遮蔽，</span></p>
<p><span style="font-size: 12pt; color: #000000;">真是超級消暑聖地呀~</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_223201_925" src="https://pimg.1px.tw/lolwarden/1467719220-3328193767.jpg" alt="IMG_20160703_223201_925" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">還有遊客特地帶椅子來這裡坐，超專業!</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_223207_312" src="https://pimg.1px.tw/lolwarden/1467719220-722947242.jpg" alt="IMG_20160703_223207_312" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">假裝要跳水嗎 哈哈哈&nbsp;</span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_224306_212" src="https://pimg.1px.tw/lolwarden/1467719220-406575182.jpg" alt="IMG_20160703_224306_212" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img style="color: #0000ff; font-family: 新細明體; letter-spacing: 1px; line-height: 20px;" title="IMG_20160703_223150_227" src="https://pimg.1px.tw/lolwarden/1467719300-2090645136.jpg" alt="IMG_20160703_223150_227" border="0"></span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;"><span style="letter-spacing: 1px; line-height: 20px; font-family: 新細明體;">這裡的溪水真的超乾淨，可以</span><span style="letter-spacing: 1px; line-height: 20px; font-family: 新細明體;">讓有潔癖的我直接光腳踩XDD</span></span></p>
<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img style="color: #0000ff; font-family: 新細明體; letter-spacing: 1px; line-height: 20px;" title="IMG_20160703_224328_492" src="https://pimg.1px.tw/lolwarden/1467719300-1518075799.jpg" alt="IMG_20160703_224328_492" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">冰涼清澈溪水，直接光腳踩上去腳也不會痛喔</span></p>
<p><span style="font-size: 12pt; color: #000000;"><img style="color: #0000ff; font-family: 新細明體; letter-spacing: 1px; line-height: 20px;" title="IMG_20160703_224523_095" src="https://pimg.1px.tw/lolwarden/1467719220-1603458498.jpg" alt="IMG_20160703_224523_095" border="0">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>

<p><span style="letter-spacing: 1px; line-height: 20px; color: #000000; font-family: 新細明體; font-size: 12pt;"><img title="IMG_20160703_222925_104" src="https://pimg.1px.tw/lolwarden/1467719300-3258928861.jpg" alt="IMG_20160703_222925_104" border="0"></span></p>

<p><span style="font-size: 12pt; color: #000000;">這裡真的是個很美的地方<img src="//s.pixfs.net/f.pixnet.net/images/emotions/heart.gif" alt="" border="0"></span></p>
