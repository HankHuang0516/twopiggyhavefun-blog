<p>&nbsp;</p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">剛好我上班地點就在遼寧夜市附近，是個充滿許多小吃、飲料店的集合地~</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">今天跟大家分享這家我每到冬天必吃的「<span style="color: #ff0000;"><strong>阿圖麻油雞</strong></span>」</span></p>
<p><span style="font-size: 12pt;"><img title="IMG_20160215_222552_127" src="https://pimg.1px.tw/lolwarden/1456060252-537309632.jpg" alt="IMG_20160215_222552_127" border="0"></span></p>
<p><span style="font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">位置非常好找，比較靠近長安東路的那邊，而且招牌非常顯眼，上面還有一隻雞的標誌</span></p>
<p><span style="font-size: 12pt;"><img title="IMG_20160215_222555_712" src="https://pimg.1px.tw/lolwarden/1456060252-2507923421.jpg" alt="IMG_20160215_222555_712" border="0"></span></p>
<p><span style="font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-family: arial; font-size: 12pt; letter-spacing: 1px; line-height: 20px; color: #000000;">阿圖麻油雞菜單</span></p>
<p><span style="font-size: 12pt;"><img title="IMG_20160215_222546_648" src="https://pimg.1px.tw/lolwarden/1456060252-3575989242.jpg" alt="IMG_20160215_222546_648" border="0"></span></p>
<p><span style="font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>麻油乾麵線 $35元</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">這樣份量一人吃剛剛好，麻油香氣濃郁，完全吸附在麵條上</span></p>
<p><span style="font-size: 12pt; color: #000000;">是我每次來必點的麵線~</span></p>
<p><span style="font-size: 12pt; color: #000000;">這回帶男友一起來吃，也大讚不錯唷!!</span></p>
<p><span style="font-size: 12pt;"><img title="IMG_20160215_222534_190" src="https://pimg.1px.tw/lolwarden/1456060252-467512927.jpg" alt="IMG_20160215_222534_190" border="0"></span></p>
<p><span style="font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>麻油雞 $120　</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">裡面有一塊腿、一塊背、一塊雞胸</span></p>
<p><span style="font-size: 12pt; color: #000000;">濃濃酒香加上香氣撲鼻的麻油香氣，喝下一口，立刻全身暖呼呼</span></p>
<p><span style="font-size: 12pt; color: #000000;">這次米酒味道不知道為什麼有點重，就沒有全部喝完，但是雞肉非常鮮嫩好吃!!</span></p>
<p><span style="font-size: 12pt;"><img title="IMG_20160215_222522_830" src="https://pimg.1px.tw/lolwarden/1456060252-1921447048.jpg" alt="IMG_20160215_222522_830" border="0"></span></p>
<p><span style="font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>青菜 $30元</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">青菜口感一般</span></p>
<p><span style="font-size: 12pt;"><img title="IMG_20160215_222538_950" src="https://pimg.1px.tw/lolwarden/1456060252-2314117842.jpg" alt="IMG_20160215_222538_950" border="0"></span></p>
<p><span style="font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #ff6600;"><strong>乾鮮蚵 $70元</strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">新鮮的蚵仔加上醬油及薑絲就非常好吃，相當肥美飽滿</span></p>
<p><span style="font-size: 12pt;"><img title="IMG_20160215_222542_995" src="https://pimg.1px.tw/lolwarden/1456060252-4022429853.jpg" alt="IMG_20160215_222542_995" border="0">&nbsp;&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">雖然遼寧夜市不大，不過還是有很多好吃的美食~</span></p>
<p><span style="font-size: 12pt; color: #000000;">阿圖麻油雞即使用中午用餐時間來，仍然人潮滿滿~&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">天冷冷時，就來阿圖麻油雞暖暖身吧!</span></p>
