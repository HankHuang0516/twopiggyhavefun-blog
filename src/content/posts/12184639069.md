<p><span style="font-size: 12pt;"><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; color: #0000ff;">** 喜歡旅遊、愛吃美食嗎？歡迎加入&nbsp;</span><span style="color: #777777; font-family: arial; letter-spacing: 1px; line-height: 20px; text-decoration: underline;"><a style="color: #000000; text-decoration: underline;" href="https://www.facebook.com/twopiggyhavefun/?fref=ts" target="_blank"><span style="color: #ff0000; text-decoration: underline;">兩隻小豬到處吃喝玩樂遊記</span><span style="color: #ff0000; text-decoration: underline;">粉絲團</span></a></span><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; color: #0000ff;">**</span></span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="page" src="https://pimg.1px.tw/lolwarden/1452054843-2662863243.jpg" alt="page" border="0"></span></p>
<p><span style="font-size: 12pt;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">上次吃跳舞香水已經是4年前的事了，吃過一次後就對它的蜜糖吐司念念不忘啊~</span></p>
<p><span style="font-size: 12pt; color: #000000;">剛好跨年那天提早下班，就相約我的好姊妹+好男友來吃下午茶囉~</span></p>
<p><span style="font-size: 12pt; color: #000000;">這天我們來到跳舞香水忠孝店，店家非常好找，招牌也非常醒目，就在東區頂好對面!!</span></p>
<p><span style="color: #000000;"><span style="font-size: 12pt;">跳舞香水不只提供女孩們愛吃的甜點，還有義大利麵、燉飯、薄餅...即使帶男生來他們也能吃飽飽唷，</span><span style="font-size: 12pt;">店內還有各種面具可以讓客人玩，在等待餐點送來時，拿來試戴也不錯歐</span></span></p>
<p>&nbsp;</p>
<p><span style="color: #ff6600;"><strong><span style="font-size: 12pt;">蜂蜜鬆餅&nbsp;</span></strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">總共有4塊鬆餅、1小杯蜂蜜，可以隨個人喜好加~</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="IMG_20151231_204523_034" src="https://pimg.1px.tw/lolwarden/1452054843-1902541235.jpg" alt="IMG_20151231_204523_034" border="0"></span></p>
<p>&nbsp;</p>
<p><span style="font-size: 12pt; color: #000000;">鬆餅還有附上擠花奶油，怕胖的人就不要沾太多哈哈</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="蜂蜜鬆餅" src="https://pimg.1px.tw/lolwarden/1452054843-3795248846.jpg" alt="蜂蜜鬆餅" border="0"></span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">不得不跟大家說~~這鬆餅的厚度至少有3公分耶</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">而且口感很扎實，一個人吃完就非常飽</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="IMG_20151231_204531_724" src="https://pimg.1px.tw/lolwarden/1452054843-725089769.jpg" alt="IMG_20151231_204531_724" border="0"></span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="color: #ff6600;"><strong><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt;">黑金巧克力</span></strong></span></p>
<p><span style="font-size: 12pt; color: #000000;"><span style="font-family: arial; letter-spacing: 1px; line-height: 20px;">哇~~擺盤的相當美!!還有跳舞香水英文字的藝術簽名，蜜糖吐司</span><span style="font-family: arial; letter-spacing: 1px; line-height: 20px;">一塊塊堆疊上去，加上一球巧克力冰淇淋，讓人不知該從何吃起~~</span></span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="黑金巧克力" src="https://pimg.1px.tw/lolwarden/1452054843-3843661891.jpg" alt="黑金巧克力" border="0"></span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">個人非常喜歡蜜糖吐司的鬆軟口感，加上巧克力冰淇淋一起吃，雙重口感~~水滴巧克力也不錯吃!!</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="IMG_20151231_204538_129" src="https://pimg.1px.tw/lolwarden/1452054843-107506309.jpg" alt="IMG_20151231_204538_129" border="0"></span></p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p><span style="color: #ff6600;"><strong><span style="font-size: 12pt;">粉紅名媛蜜糖吐司</span></strong></span></p>
<p><span style="font-size: 12pt; color: #000000;">吐司切成方形，外層披上糖衣，最上層是冰淇淋，再淋上奶油、草莓醬，充滿視覺與味蕾</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="粉紅名媛金磚吐司" src="https://pimg.1px.tw/lolwarden/1452054843-3641796771.jpg" alt="粉紅名媛金磚吐司" border="0"></span></p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p><span style="font-size: 12pt; color: #000000;">草莓薄餅加上水果茶，真是幸福滋味~~</span></p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;"><img title="201511_跳舞香水_鬆餅(Nota) (2)" src="https://pimg.1px.tw/lolwarden/1452054843-516879075.jpg" alt="201511_跳舞香水_鬆餅(Nota) (2)" border="0"> &nbsp;</span></p>
<p>&nbsp;</p>
<p><span style="font-family: arial; letter-spacing: 1px; line-height: 20px; font-size: 12pt; color: #000000;">吃飽後呈現一個放鬆狀態，我好喜歡這裡的沙發座唷~~</span></p>
