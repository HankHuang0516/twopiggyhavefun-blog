<p><span style="font-size: 12pt; color: #000000;">活動時間：2016年9月1日(週四)～9月11日(週日)</span></p>
<p><span style="font-size: 12pt; color: #000000;">週末露天電影院：2016年9月2日(週五)、3日(週六)、4日(週日)、9日(週五)、10日(週六)、11日(週日)</span></p>
<p><span style="font-size: 12pt; color: #000000;">6：30PM 開放JCB卡持卡人優先入場</span></p>

<p><span style="font-size: 12pt; color: #000000;">7：00PM 放映開始</span></p>

<p><span style="letter-spacing: 1px; line-height: 20px; font-family: 新細明體; color: #000000; font-size: 12pt;"><img title="IMG_8007" src="https://pimg.1px.tw/lolwarden/1473249435-4233492230.jpg" alt="IMG_8007" border="0"></span></p>
<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
<p><span style="font-size: 12pt; color: #000000;">小時候每天常看的卡通就是「哆啦A夢」啦~~</span></p>
<p><span style="font-size: 12pt; color: #000000;">相信許多人小時候都想要有個哆啦A夢的任意門及百寶袋</span></p>
<p><span style="font-size: 12pt; color: #000000;">現在就可以使用任意門來到中正紀念堂，感受快樂無憂童年!</span></p>
<p><span style="font-size: 12pt; color: #000000;">自即日起至11日止，中正紀念堂的民主大道廣場中</span></p>
<p><span style="font-size: 12pt; color: #000000;">打造超<span style="font-family: Arial, Helvetica, sans-serif; line-height: 21.248px;">過600平方公尺的「哆啦A夢」裝置藝術</span></span></p>
<p><span style="font-size: 12pt; color: #000000;">歡迎大小朋友一同來現場拍照同樂</span></p>

<p><span style="font-size: 12pt; color: #000000;">&nbsp;</span></p>
